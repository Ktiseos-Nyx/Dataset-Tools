Development notes :

> - Dev Installation:
> ```
> pip install -e .
> ```

> - Reinstallation:
> ```
> pip uninstall dataset-tools
> ```

> - Update version:
> ```
> python -m _version
> ```

>
> ```
> - Run with debug logging :
> ```
> dataset-tools -d
> ```

> or

> ```
> python -m main --log debug
> ```
